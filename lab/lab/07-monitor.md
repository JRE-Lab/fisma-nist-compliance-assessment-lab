+# Lab 07: Monitor (RMF Step 6)
+
+## Purpose
+
+Establish continuous monitoring to ensure ongoing control effectiveness.
+
+## Objectives
+
+- Define a continuous monitoring strategy.
+- Identify metrics and reporting cadence.
+- Integrate vulnerability management and incident response.
+
+## Tasks
+
+1. Identify key controls for ongoing assessment.
+2. Define metrics (e.g., patching SLA, log review frequency).
+3. Draft a continuous monitoring strategy.
+4. Update the SSP with monitoring responsibilities.
+
+## Deliverables
+
+- Continuous monitoring strategy
+- Metrics and reporting schedule
+- Updated SSP monitoring section
+
+## Tips
+
+- Align metrics with the system's impact level.
+- Include dependencies such as cloud provider reports.
